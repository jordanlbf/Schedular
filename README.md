# Schedular

POS software with enhanced AI scheduling for delivery services.

---

## 🚀 Quickstart

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
uvicorn app:app --reload --port 8000
```

### One command

* **Windows:**

  * In **PowerShell**:

    ```powershell
    .\run-dev.ps1
    ```
  * In **CMD Prompt**:

    ```bat
    run-dev.bat
    ```

    or

    ```bat
    powershell -ExecutionPolicy Bypass -File run-dev.ps1
    ```

* **macOS/Linux:**

  ```bash
  ./run-dev.sh
  ```

## 📂 Project Layout

* `frontend/` — React + Vite + TypeScript
* `backend/` — FastAPI
* `docs/` — documentation and notes
